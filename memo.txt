VS CodeからGoogle Apps Script (GAS) にコードをアップロードするには、Googleが提供しているCLIツール **「clasp」** を使用するのが一般的です。

アップロードされたファイルに `.clasp.json` が含まれているため、このプロジェクトは既に `clasp` で管理できる状態になっています。

以下の手順でコマンドを実行してください。

### 1. 事前準備

PCに **Node.js** がインストールされている必要があります。
ターミナルで `node -v` と入力してバージョンが表示されればOKです。未インストールの場合は公式サイトからインストールしてください。

### 2. Claspのインストール（初回のみ）

VS Codeのターミナルを開き、以下のコマンドで `clasp` をインストールします。

```bash
npm install -g @google/clasp

```

### 3. Googleアカウントにログイン

GASにアクセス権限を与えるため、ログインします。ブラウザが開くので承認してください。

```bash
clasp login

```

### 4. ディレクトリの移動

`.clasp.json` があるディレクトリ（フォルダ）でコマンドを実行する必要があります。
現在のファイル構成では、以下のディレクトリに移動してください。

```bash
cd mizuno1224/club-attendance-gas/club-attendance-gas-c0482763e912ada9febd4adf7c6aa3328df2e9e5/

```

※フォルダ名はご自身の環境に合わせて調整してください。`.clasp.json` がある場所がルートになります。

### 5. コードをアップロード（Push）

以下のコマンドを実行すると、ローカルのコードがGASプロジェクトにアップロードされます。

```bash
clasp push

```

成功すると `Pushed 〇 files.` と表示されます。

---

### 便利なコマンド

* **変更を監視して自動アップロード**:
開発中は以下のコマンドを実行しておくと、ファイルを保存するたびに自動でGASにアップロードされます。
```bash
clasp push --watch

```


* **GAS上の変更をローカルに取り込む（Pull）**:
ブラウザのエディタ側で変更を加えた場合は、上書きを防ぐためにローカルに取り込みます。
```bash
clasp pull

```


* **ブラウザでGASエディタを開く**:
```bash
clasp open

```
